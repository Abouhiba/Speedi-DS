Color-coded campaign tag for Speedi's promotional system — use it on product cards, PDP, cart rows and campaign pages; one component covers all campaign families.

```jsx
<PromoTag kind="bogo" />
<PromoTag kind="percent" label="35% OFF" solid />
<PromoTag kind="wheel" size="sm" />
```

Families: percent/flash (red), bogo/bundle (violet), coupon/cashback (blue), sample/gift (pink), wheel/points (gold), freeDelivery/fresh (green). `solid` for placement over imagery; tinted default for placement on white.
